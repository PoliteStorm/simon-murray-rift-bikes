## Important Notes

⚠️ **Legal Compliance:**
- Always check brand usage guidelines
- Some brands require permission for commercial use
- Follow brand guidelines for colors and spacing
- Do not modify official logos

✅ **Best Practices:**
- Use SVG format when available (scalable)
- Use high-resolution PNG if SVG not available
- Maintain original aspect ratios
- Keep logos in their official colors

## Verification

After downloading:
1. ✅ Check file is SVG or high-res PNG
2. ✅ Verify logo displays correctly
3. ✅ Ensure you have usage rights
4. ✅ Test on website

## Need Help?

If you can't find logos:
1. Contact brand's marketing department
2. Check Wikimedia Commons
3. Look for brand asset repositories
4. Check if brand has a partner/reseller portal
