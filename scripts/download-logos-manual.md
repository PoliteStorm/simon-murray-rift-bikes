# Official Brand Logo Download Guide

## How to Download Official Logos

### Method 1: Official Brand Websites

Visit each brand's official website and look for:
- **Media Kit** section
- **Press Resources** 
- **Brand Assets**
- **Downloads**

### Method 2: Wikimedia Commons (Public Domain/CC Licensed)

Many brand logos are available on Wikimedia Commons:
- https://commons.wikimedia.org/wiki/Category:Logos_of_bicycle_brands

### Method 3: Direct Download URLs

#### Shimano
- Wikimedia: https://upload.wikimedia.org/wikipedia/commons/8/8a/Shimano_logo.svg
- Official: https://www.shimano.com (check Media section)

#### SRAM
- Official: https://www.sram.com (check Media/Press section)
- Wikimedia: Search "SRAM logo"

#### Campagnolo
- Official: https://www.campagnolo.com (check Corporate/Media section)
- Wikimedia: Search "Campagnolo logo"

#### Maxxis
- Official: https://www.maxxis.com (check Media resources)

#### RockShox
- Official: https://www.rockshox.com (check Brand Assets)

#### JAGWIRE
- Official: https://jagwire.com (contact for brand assets)

#### Bafang
- Official: https://www.bafang-e.com (check Media section)

#### Tektro
- Official: https://www.tektro.com (check Brand Resources)

## Download Instructions

1. **Find the logo** on the official website or Wikimedia Commons
2. **Download SVG format** (preferred) or high-resolution PNG
3. **Save to**: `/home/tau/RIFT/public/logos/partners/`
4. **Rename** to match the component name:
   - `shimano.svg`
   - `sram.svg`
   - `campagnolo.svg`
   - `maxxis.svg`
   - `rockshox.svg`
   - `jagwire.svg`
   - `bafang.svg`
   - `tektro.svg`

## Important Notes

⚠️ **Always verify you have permission to use brand logos!**
- Check each brand's usage guidelines
- Some brands require permission for commercial use
- Follow brand guidelines for colors, spacing, and placement

## Quick Download Commands

```bash
# Navigate to logos directory
cd /home/tau/RIFT/public/logos/partners

# Download Shimano from Wikimedia (example)
curl -L "https://upload.wikimedia.org/wikipedia/commons/8/8a/Shimano_logo.svg" -o shimano.svg

# For other brands, visit their official websites
```

## Verification

After downloading, verify:
- ✅ Logo is in SVG or high-res PNG format
- ✅ File is named correctly
- ✅ Logo displays properly on website
- ✅ You have permission to use it
