# RIFT Marketing Logos

A collection of logo variations for marketing materials, posters, and video overlays.

## Logo Files

All logos use the royal green and gold color scheme with transparent backgrounds (perfect for overlaying).

### 1. **rift-logo-horizontal.svg**
- Horizontal layout with icon and text
- Best for: Website headers, email signatures, letterheads
- Dimensions: 400x120px

### 2. **rift-logo-vertical.svg**
- Vertical stacked layout
- Best for: Mobile apps, social media profiles, square formats
- Dimensions: 200x300px

### 3. **rift-logo-icon-only.svg**
- Icon only, no text
- Best for: Favicons, app icons, small placements
- Dimensions: 200x200px

### 4. **rift-logo-minimal.svg**
- Clean, minimalist design
- Best for: Subtle branding, watermarks, minimal designs
- Dimensions: 300x100px

### 5. **rift-logo-badge.svg**
- Badge style with emerald green background
- Best for: Certificates, achievements, premium branding
- Dimensions: 300x300px

### 6. **rift-logo-stacked.svg**
- Icon above text with tagline
- Best for: Business cards, marketing materials
- Dimensions: 250x250px

### 7. **rift-logo-wide-banner.svg**
- Wide banner format with tagline
- Best for: Website headers, banners, wide format posters
- Dimensions: 800x200px

### 8. **rift-logo-circle-badge.svg**
- Circular badge design
- Best for: Profile pictures, round formats, badges
- Dimensions: 300x300px

### 9. **rift-logo-text-only.svg**
- Text only version with glow effect
- Best for: Text-based branding, when icon isn't needed
- Dimensions: 400x150px

### 10. **rift-logo-compact.svg**
- Compact version for small spaces
- Best for: Mobile headers, compact spaces
- Dimensions: 180x60px

### 11. **rift-logo-watermark.svg**
- Subtle watermark version
- Best for: Video overlays, image watermarks, subtle branding
- Dimensions: 200x200px

## Color Scheme

- **Gold**: #fbbf24 (Primary accent)
- **Emerald Green**: #10b981 (Secondary accent)
- **Royal Green**: #0d4d3f (Background when needed)
- **White**: #ffffff (Text)

## Usage Tips

- All logos have transparent backgrounds
- SVG format scales perfectly to any size
- Can be converted to PNG/JPG for specific use cases
- Perfect for overlaying on images and videos
- Use watermark version for subtle branding

## For Video/Image Overlays

Recommended logos for overlaying:
- `rift-logo-watermark.svg` - Subtle watermark
- `rift-logo-icon-only.svg` - Corner placement
- `rift-logo-compact.svg` - Bottom corner
- `rift-logo-text-only.svg` - Text overlay
