# RIFT Bikes - Custom Bike Shop

**Simon Murray - Bikes for Charity**

Premium custom road bikes, locally assembled with a percentage of profits donated to local charities.

## Contact

- **Email:** si@riftbike.com
- **Phone:** 07817174391

## Features

- Custom bike builds
- Bank transfer payment system
- Locally assembled bikes
- Lifetime frame warranty
- 1 year warranty on all components
- Charity donations from profits

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- SQLite Database

## Setup

```bash
npm install
npm run dev
```
